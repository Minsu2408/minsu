알고리즘 기초 - 반복과 재귀
팩토리얼 함수
반복적 구현 예시

python
def factIter(n):
    v = 1
    for i in range(1, n+1):
        v *= i
    return v
재귀적 구현 예시

python
def factRecur(n):
    if n == 0:
        return 1
    return n * factRecur(n-1)

반복적 방법

python
def addOneIter(L):
    for i in range(len(L)):

재귀적 방법

python
def addOneRecur(L, i):
    if i == len(L):
        return
    L[i] += 1
    addOneRecur(L, i+1)
리스트 원소 탐색
반복적 탐색

python
def searchIter(L, key):
    for i in range(len(L)):
        if L[i] == key:
            return i
    return -1
재귀적 탐색

python
def searchRecur(L, key, i):
    if i == len(L):
        return -1
    if L[i] == key:
        return i
    return searchRecur(L, key, i+1)
리스트 합 구하기
반복적 합산

python
def sumIter(L):
    total = 0
    for i in range(len(L)):
        total += L[i]
    return total
재귀적 합산

python
def sumRecur(L, i):
    if i == len(L):
        return 0
    return L[i] + sumRecur(L, i+1)
리스트 최댓값 찾기
반복적 최댓값

python
def greatestIter(L):
    currentMax = L[0]
    for i in range(len(L)):
        if L[i] > currentMax:
            currentMax = L[i]
    return currentMax
재귀적 최댓값

python
def greatestRecur(L, i):
    if i == len(L) - 1:
        return L[i]
    return max(L[i], greatestRecur(L, i+1))
